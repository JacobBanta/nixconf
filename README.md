# nixconf - (WIP)
This is my flake based NixOS configuration.
I'm pretty sure it classifies as [dendritic](https://github.com/mightyiam/dendritic).
You can use my programs by running:
```bash
nix run github:jacobbanta/nixconf#<program>
```

I have the following programs configured
* `neovim` - very basic neovim setup
* `ghostty` - base ghostty with fish set as the shell
* `hyprland` - basically default hyprland
* `hyprlandNested` - same as `hyprland`, but $MOD is ALT so it doesnt interfere with your WM.
